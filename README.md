# Flask Hotel Booking App

![фото](https://avatars.mds.yandex.net/get-altay/11398069/2a000001917a400321157583a2def53c405e/XXXL)

## Описание
Это веб-приложение для бронирования номеров в отелях, разработанное на фреймворке Flask. Пользователи могут просматривать доступные номера, бронировать их, регистрироваться и управлять заказами. Проект создан в рамках курсовой работы по веб-разработке.

## Технологии
- Backend: Flask (Python)
- База данных: PostgreSQL
- Фронтенд: HTML/CSS с Jinja2 шаблонами, Bootstrap
- Дополнительно: SQLAlchemy для ORM, WTForms для форм
